# AbsenPanha Flutter WebView App

## Run

```bash
flutter pub get
flutter run
```

## Notes

- URL default: `https://absenpanha.com`
- Android back button:
  - Go back in WebView history if possible
  - Exit app if no history
- Loading indicator shown while page is loading
- Uses SafeArea
- Android `minSdkVersion` set to `19`
- iOS deployment target set to `11.0`
